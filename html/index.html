CHIP-8 Emulator

Controls

The CHIP-8 uses a 16-key hexadecimal keypad mapped to your keyboard:

1 2 3 4 → 1 2 3 C
Q W E R → 4 5 6 D
A S D F → 7 8 9 E
Z X C V → A 0 B F

Debugger

F5 - Pause
F6 - Step
F8 - Resume

About

This is a CHIP-8 interpreter compiled to WebAssembly using Emscripten. CHIP-8 is an interpreted programming language developed in the 1970s for early microcomputers.
